# CampusOverflow (clonelike)

This is a collaborative campus developer platform built with React and Vite.

## Tech Stack
- **Frontend**: React, Vite, Lucide-React
- **Styling**: Vanilla CSS (Modern Indigo Theme)
- **Data**: Hybrid feed (StackOverflow API + Campus Mocks)

## Getting Started
1. `npm install`
2. `npm run dev`
